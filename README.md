# Boot-Camp-1 (Python Exception Handling & Inheritance)

A revision repository focused on demonstrating the various facets of **Exception Handling** and **Inheritance** in Python. 

## 🚀 Topics Covered

- **Built-in Exceptions**: Handling standard Python errors such as `ZeroDivisionError` and `NameError` (`Built-in.py`).
- **Raising Exceptions**: Throwing exceptions manually using the `raise` keyword when specific conditions are met (`RaisingandHandlingExceptions.py`).
- **Custom Exceptions**: Creating and handling tailored exceptions (like `ValueError`) for specific application logic constraints (`CustomExceptionHandling.py`).
- **Nested Exception Handling**: Demonstrating the flow of `try`, `except`, `else`, and `finally` blocks in a nested format to gracefully recover from consecutive potential points of failure (`NestedExceptionHandling.py`).

## 🛠 Prerequisites

- Python 3.x installed on your machine.

## 📝 Running the Scripts

To execute any script in this project, use the standard python command:
```bash
python CustomExceptionHandling.py
```

## 🧩 Overview of Exception Blocks used:
- `try`: Lets you test a block of code for errors.
- `except`: Lets you handle the error.
- `else`: Lets you execute code when there is no error.
- `finally`: Lets you execute code, regardless of the result of the try- and except blocks.
